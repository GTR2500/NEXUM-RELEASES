# Release Policy

## Scope

This repository is a public stable release channel for Nexum.

## Single stable release rule

Only the current stable release is published here.  
Older releases are not retained publicly.

## Replacement rule

Publishing a new stable release supersedes the previous one.

## Versioning

SemVer is used: `MAJOR.MINOR.PATCH`

## Integrity

Each published release must provide

- installer artifacts
- integrity hashes (SHA256 recommended)
- concise release notes describing impacted modules and compatibility

## Non goals

This repository is not a public archive of previous versions.
